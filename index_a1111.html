Core Concepts

How Stable Diffusion Works

  • Diffusion Process: Model learns to reverse noise into images
  • Latent Space: Compressed image representation (smaller than pixels)
  • Text Encoder: CLIP converts prompts into embeddings
  • U-Net: Neural network that predicts and removes noise
  • VAE: Decodes latent representation into final image

Generation Process

  1. Text prompt → CLIP embeddings
  2. Random noise in latent space
  3. Iterative denoising (sampling steps)
  4. VAE decodes to image
  5. Output image

💡 Key Insight

Higher steps = more iterations = better quality but slower generation. Diminishing returns after 30-50 steps.

Essential Terminology

Term Meaning
Checkpoint Main model file (.ckpt/.safetensors). The "brain" of SD.
VAE Variational AutoEncoder. Affects colors/brightness. Usually baked into models now.
LoRA Low-Rank Adaptation. Small addon models for styles, characters, concepts.
Embedding Textual Inversion. Trigger word that represents a concept.
HyperNetwork Older training method. Like LoRA but modifies network structure.
CFG Scale Classifier-Free Guidance. How closely to follow prompt (7-12 typical).
Seed Random number for reproducibility. Same seed + settings = same image.
Resolution Image size. Must be multiples of 8. SD 1.5: 512x512, SDXL: 1024x1024.
Steps Sampling iterations. 20-30 for speed, 30-50 for quality.
Prompt Text description of desired output.
Negative Prompt What to avoid in generation.
Upscale Increase resolution using AI (e.g., ESRGAN, SwinIR).
Inpainting Edit specific parts of an image.
Outpainting Extend image beyond original borders.
ControlNet Extension for precise control (pose, depth, edges).

Interface Tabs

txt2img

Generate images from text prompts. Main tab for creating from scratch.

img2img

Transform existing images. Uses image + text prompt as guidance.

  • Denoising Strength: How much to change (0.0 = same, 1.0 = completely new)
  • 0.3-0.5: Small variations
  • 0.6-0.75: Moderate changes
  • 0.75-0.95: Major changes

Inpaint

Edit specific regions. Paint mask over area to change.

  • Inpaint masked: Change painted area
  • Inpaint not masked: Keep painted area, change rest
  • Mask blur: Softness of mask edges (4-12 typical)
  • Masked content: Latent noise/fill/original

Extras

Upscale, restore faces, remove backgrounds.

PNG Info

Extract generation parameters from image metadata.

Checkpoint Merger

Blend multiple models together.

Train

Create embeddings, hypernetworks, or prepare datasets.

Prompt Engineering

Prompt Structure

(quality tags), (subject), (action/pose), (clothing), (environment), (lighting), (style), (artist), (technical specs)

Token Priority

Earlier tokens have more weight. Put most important descriptors first.

Attention Syntax

Syntax Effect Example
(word) 1.1x weight (masterpiece)
((word)) 1.21x weight (1.1²) ((best quality))
[word] 0.91x weight (reduce) [lowres]
(word:1.5) Exact 1.5x weight (detailed:1.3)
[word:0.5] Exact 0.5x weight [blurry:0.3]

Scheduled Prompting

Change prompt during generation:

[from:to:when] Example: [sunset:dusk:0.3] → Starts with "sunset", switches to "dusk" at 30% of steps Alternate: [cat|dog:0.5] → Alternates between cat and dog every 0.5 steps

Quality Boosters

masterpiece, best quality, highres, absurdres, incredibly absurdres, ultra-detailed, intricate details, professional, 8k uhd, RAW photo

Common Negative Prompts

lowres, bad anatomy, bad hands, text, error, missing fingers, extra digit, fewer digits, cropped, worst quality, low quality, normal quality, jpeg artifacts, signature, watermark, username, blurry, artist name, mutation, deformed, ugly, duplicate, morbid, mutilated, out of frame, disfigured

Common Workflows

🖼️ Character Portrait

  1. Base model: Realistic Vision / Juggernaut
  2. Resolution: 512x768 (SD 1.5) or 832x1216 (SDXL)
  3. Sampler: DPM++ 2M Karras
  4. Steps: 25-30
  5. CFG: 7-8
  6. Highres fix: 0.5 denoising, 2x upscale

🎨 Illustration/Anime

  1. Base model: MeinaMix / Counterfeit / Animagine XL
  2. Resolution: 512x512 or 768x512
  3. Sampler: Euler a
  4. Steps: 28
  5. CFG: 7-9
  6. Add detail LoRA at 0.6-0.8

🔄 Img2img Variations

  1. Drop image into img2img
  2. Set denoising: 0.4-0.6
  3. Adjust prompt to desired changes
  4. Keep same seed for consistency
  5. Batch count for variety

✏️ Inpainting Workflow

  1. Go to Inpaint tab
  2. Upload image
  3. Paint mask over area to change
  4. Write prompt for replacement
  5. Masked content: "Original" or "Latent noise"
  6. Mask blur: 4-8
  7. Denoising: 0.75-0.95

📈 Upscaling Workflow

  1. Generate at native resolution first
  2. Use Hires. fix in txt2img (slower but integrated)
  3. Or use Extras tab (faster, more control)
  4. Upscalers: 4x-UltraSharp, ESRGAN_4x, R-ESRGAN 4x+
  5. For faces: Enable GFPGAN or CodeFormer

👤 Face Swap Workflow

  1. Install ReActor or Roop extension
  2. Upload source face in extension panel
  3. Generate with target prompt
  4. Adjust blend strength

Installing Extensions

  1. Go to Extensions tab → Install from URL
  2. Paste GitHub repository URL
  3. Click Install
  4. Go to Installed tab → Apply and restart UI
  5. Some require additional models (check extension docs)

⚠️ Important

After installing extensions, always restart the entire WebUI (not just "Reload UI"). Some extensions need full restart to load dependencies.

Common Issues

Problem Solution
CUDA out of memory Enable "Low VRAM mode" in settings. Use --medvram or --lowvram flags
Black images Check VAE is loaded. Try FP32 precision in settings
Green tint Wrong VAE. Use VAE baked into model or matching VAE
Blurry faces Enable After Detailer or use face restoration (GFPGAN/CodeFormer)
Slow generation Use xFormers. Enable "Full precision" if on 10xx cards
Can't load model Check file isn't corrupted. Ensure enough RAM/VRAM
NSFW blocked Disable "Filter NSFW content" in settings

Launch Flags

Add to webui-user.bat (Windows) or webui-user.sh (Linux):

set COMMANDLINE_ARGS=--xformers --medvram --autolaunch --update-check

Common Flags

Flag Purpose
--xformers Faster generation, less VRAM (install first)
--medvram VRAM optimization for 8GB cards
--lowvram For 4GB cards (slower)
--precision full Fixes black images on some GPUs
--no-half Use FP32 instead of FP16
--autolaunch Open browser automatically
--api Enable REST API for external tools
--share Generate public gradio URL
